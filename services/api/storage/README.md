# Runtime Storage

This directory contains local runtime files generated by NoteFlow.

It is intentionally ignored by Git.

## Layout

```text
uploads/{document_id}.pdf
rendered/{document_id}/page-XXX.png
regions/{document_id}/page-XXX-region-YY.png
```

## Source Of Truth

PostgreSQL is the source of truth for file references:

| Table | File field |
|---|---|
| `documents` | `storage_path` |
| `document_page_assets` | `image_path` |
| `document_visual_regions` | `asset_path` |

Do not manually move UUID-named runtime folders unless the database paths are migrated at the same time.

## Experiments

Temporary benchmark renders or crops should not be stored here permanently. Keep durable research summaries under:

```text
docs/research/
```
